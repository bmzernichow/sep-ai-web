Overview of agile AI frameworks

While agile software frameworks such as Scrum have become the de facto way of developing and delivering software today, these frameworks do not necessarily translate well when applied to data science projects. There is a fundamental difference between delivering software products and delivering actionable insights from data as project results. To support data science projects, new agile frameworks have emerged and some existing agile frameworks have been adjusted accordingly. Today a few agile AI frameworks exists.
